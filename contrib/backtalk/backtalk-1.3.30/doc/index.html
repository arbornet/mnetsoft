The following documentation is included in this distribution:

  • License. Standard Gnu Public License.

  • Backtalk Glossary - Terms and definitions for Backtalk.

  • Installation Manual - Detailed step-by-step instructions on how to install Backtalk on your system.

  • Customizing Backtalk Authentication - For people who want to modify Backtalk to share user logins and passwords with other web packages.

  • Administration Manual - Detailed descriptions of procedures for performing basic Backtalk administrative actions either directly from the Backtalk web interface, or from Unix. Topics include:
    • Conference Creation.
    • Conference Deletion.
    • Changing Fairwitnesses.
    • Opening and Closing Conferences.
    • Editing Conference Items.
    • User Account Creation.
    • User Validation and Invalidation.
    • Changing User Passwords.
    • User Deletion.

  • Script Language Reference Manual - A reference manual for the Backtalk script language, useful to those who want to write or customize Backtalk scripts.

  • Script Language Tutorial Manual - An introduction to Backtalk script programming.

  • Backtalk File Formats - Description of various versions of various file formats used by Backtalk. Incomplete.

  • Version History - A list of changes in Backtalk release versions.
Most user interfaces also have their own internal documentation.
