Backtalk is a sophisticated web-based computer conferencing system written and distributed by Jan Wolter and Steve Weiss. For general information about Backtalk, see the Backtalk home page. For information on installing Backtalk see the Backtalk Installation Guide.

This document describes basic Backtalk administration procedures. The Backtalk Glossary will help you with some of the terminology used here.

Most Backtalk conference administration tasks can be done either from the Unix shell or through the Backtalk web interface. This documentation describes both methods.

The following procedures are described here:

  • Conferences:
    • Creation.
    • Deletion.
    • Changing Fairwitnesses.
    • Opening and Closing.
    • Editing Items.
    • Unerasing Items.
    • Back-ups and Restores.
  • Users:
    • Creation.
    • Validation and Invalidation.
    • Changing Passwords.
    • Deletion.
