These are some things I have done with lynx that make it a bit easier
to use.  All of the following are optional--if you find they make lynx
easier to use, go for it; if not, they do no permanent damage.

Feel free to mail webmaster@cyberspace.org if you have any questions.


Selecting an editor:

    Most people have their own favorite editor.  (See the discussions
    in the info conference if you doubt this.)  Since many are available
    and I don't want to unduly influence people, I will simply let you
    pick your favorite.  However, for new people, I will recommend
    "pico" until you feel ready to learn another.

    Press "o" and you will see the options menu.
    Press "e" to select the editor.
    Backspace (if needed to erase former option).
    Type the name of your editor.
    Press your "enter" key.
    Type ">" to save it.

    Notice that if you don't save it, you can try out an editor without
    it becoming your permanent one.


Making your own menu:

    By adding bookmarks, you can make your own list of choices to see.
    Simply press "a" and answer "y" to add the file you are viewing
    to your bookmark file.


Using your own menu:

    Press "v" (to "view") and select one.

    You may edit your bookmark file by pressing "e" as you view it.


Adding your home directory:

    Follow these steps to add your home directory to your bookmark file.


Making a homepage file:

    mkhomepage is a new easy way to create a homepage for the World Wide Web.
You will receive a new prototype homepage which you can edit.  Most people
include a little bit of personal information and/or a listing of their
favorite links.

    Select this to view a homepage which was created by mkhomepage. Once there,
type "e" to edit or "q" to return to this screen.  (Warning: This
will take a looooong time to start up.)

    If you would like to learn more about HTML, the language used
to create Web pages, select here.
