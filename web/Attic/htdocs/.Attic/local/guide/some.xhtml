Some Useful Commands

    For handling items:
    read       browse     join       find
    print      respond    thaw       freeze
    check      remember   next       enter
    retire     unretire   postpone   header
  
    For handling responses:
    text       pseudonym  again      current
    pass       new        expunge    since
    first      last       kill       only

    For dealing with other things:
    help       write      resign
    display    echo       mail       change
    system     leave      bye
    define     undefine   quit       source
