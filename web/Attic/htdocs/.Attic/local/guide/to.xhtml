To Chat With Another User in Real-time

You may see a message like this while you're using GREX:
...Message from mju on tty00 at Wed Jun 2 00:12 1993 EDT
This means that user "mju" wants to talk privately with you. To respond, type:
write
Everything you type will show up on both screens.

To avoid confusion, it's customary to end each of your transmissions by typing an "o" (for "over") on its own line. Once you've done so, you should wait for the other person to signal that he or she is finished before you type again.

When you're finished with the conversation, type "oo" (for "over and out) on its own line. When the other person responds with oo, you can type ctrl-D to return to what you were doing.

To initiate a conversation, type:

!write userid
where userid is that of a GREX user currently on-line. See the !who command.
