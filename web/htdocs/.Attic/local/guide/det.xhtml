Determining Your Terminal Type

If you're using a communications program to call GREX, chances are your terminal type is 'ansi' or 'vt100'. Look at your screen for the help menus to find something like this. If you can't find anything, use the 'ansi-simple' terminal type. (Type it without the quotes marks.)

If you're calling on an old fashioned terminal and you can't find the model, type 'dumb' (without the quotes).

This will get you started, and you can always change your terminal type later using the "modify" program, if you want to.
