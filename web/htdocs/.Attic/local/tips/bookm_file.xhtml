A bookmark file is created for you the first time you add a bookmark.

You may choose the name of the file in the options menu. The default value is lynx_bookmarks.html, and it is stored in your home directory (/home/your_login).

To view your bookmark file, simply press "v". You may select from the bookmarks you have stored or edit it by pressing "e" (after you have selected an editor in the options menu).
