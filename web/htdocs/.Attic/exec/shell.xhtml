Access to Unix Shells and Development tools

If you are interested in using a Unix command shell, you may do so at any time, even if Lynx is your login shell. This is not recommended for users who are not familiar with Unix, but for those who are, it is not at all discouraged either.

All the Grex staff requests is that you be a good citizen. We all have to share this machine, and some of its resources are limited. We have seen all kinds of hacks designed to take the system down or otherwise harass users, and we are granting this access with the understanding that you are far too mature to try to do something like that with it. The shells currently installed are:

  • sh, Bourne Shell, the basic one
  • csh, C Shell
  • tcsh, Turbo C Shell, an enhanced C Shell
  • bash, Bourne-Again Shell from GNU
Remember to return to Lynx from any shell by issuing the exit command.
