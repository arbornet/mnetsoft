Editors available on Grex

There are many editors installed on Grex for you to choose from. Deciding which editor is best usually starts strong diagreements, so we'll deem it sufficient to say that each different editor has different capabilities and different people who like them.
  • There's vi, which is a nearly ubiquitous editor on unix machines. For a quick introduction, click here. To escape from it if you're trapped, type the escape character followed by :q!
  • There is GNU emacs, which is a very powerful and configurable editor. It is much loved by Unix users who are used to it. Exit with ^X^C.
  • There's pico, which stands for PIne COmposer. It is the text collector for the "pine" mailer. It has a command menu on the screen, so it is quite easy for beginners to learn.
  • There is bbsed, which is a "line editor." That means that you can enter one line of text at a time. It's a nice choice for dumb terminal software. We have a bbsed introduction to help you use it.
